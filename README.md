## 🌱 [todo-on-any-front-framework](https://github.com/gdsc-ssu/todo-on-any-front-framework) 스터디 설명

- 📌 여러 프레임워크를 사용해보고, 각각의 장단점을 경험해보기
- 📌 프레임워크 별 투두앱을 제작해보며, 각 프레임워크의 특징을 이해하기
- 📌 프로젝트 시작시에, 근거를 기반으로 프론트 기술 스택을 선택할 수 있어야 함
